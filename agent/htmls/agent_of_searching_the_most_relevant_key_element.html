Задача: нахождение наиболее релевантного ключевого узла для пользователя.

Описание: агент находит узел, над которым заданный пользователь произвёл наибольшее количество действий.

Инициализация: появление в памяти вопросной конструкции, соответствующей запросу:найти наиболее релевантный ключевой узел для пользователя

Аргументы: пользователь.

Результаты:

  • если на вход подан корректный аргумент, будет выведен наиболее релевантный узел.
  • если на вход подан не корректный аргумент, будет сгенерировано сообщение о некорректном аргументе.
